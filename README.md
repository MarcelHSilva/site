# Agendamentos Nuvem

Site de agendamentos online com design moderno e responsivo.

## 🚀 Funcionalidades

- Interface limpa e moderna
- Design responsivo para todos os dispositivos
- Botão de agendamento integrado
- Otimizado para performance

## 🛠️ Tecnologias

- HTML5
- CSS3
- Design responsivo

## 📦 Deploy

Este site está configurado para deploy automático no Railway.

### Deploy no Railway

1. Conecte este repositório ao Railway
2. O deploy será automático a cada push na branch main
3. O site estará disponível na URL fornecida pelo Railway

### Estrutura de Arquivos

```
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── .htaccess          # Configurações do servidor
└── README.md          # Documentação
```
